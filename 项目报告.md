# 中国象棋棋手风格空间构建与可视化分析

> 课程方向：高维数据可视化（High-Dimensional Visualization）
> 涉及方向：图数据可视化、可视化交互与评估

---

## 摘要

本项目从 **1724 盘**中国象棋特级大师真实对局出发，为每位棋手构建 **18 维行为特征向量**，
经标准化、PCA 降维与 K-Means 聚类，将 **20 位特级大师**投影到一张二维"风格空间地图"上，
并实现了散点地图、风格雷达、相似棋手关系网络、双人对比等交互功能。

核心结论：**象棋大师的棋风可以被量化，且在高维空间中确实存在稳定的风格流派**——
20 位大师自动聚成 5 个有意义的群体（速攻速决、平稳细腻、慢工缠斗、传统专精、残局控制），
划分与棋迷公认印象高度吻合。

---

## 一、研究背景与问题

传统棋谱分析关注胜负、开局统计与棋局回放，对"棋手风格"少有量化研究。不同棋手长期对局中
会形成稳定的行为特征（主动进攻、稳健防守、残局运营、复杂战术等），但这些特征难以直接观察。
本项目用数据分析与高维可视化技术，从棋谱中提取行为特征、构建棋手风格空间，回答：

1. **不同棋手是否存在可量化的风格差异？**
2. **高维特征空间中是否存在明显的风格聚类现象？**
3. **哪些特征最能反映棋手风格？**
4. **棋手之间是否存在风格相似关系？**

---

## 二、数据来源与处理

| 项 | 说明 |
|---|---|
| 原始数据 | 东萍象棋公开棋谱库，约 **99,771 盘** ICCS 坐标棋谱 |
| 筛选目标 | 20 位特级大师：杨官璘、胡荣华、李义庭、王嘉良、柳大华、李来群、吕钦、赵国荣、许银川、徐天红、陶汉明、于幼华、洪智、蒋川、赵鑫鑫、王天一、郑惟桐、孙勇征、谢靖、汪洋 |
| 筛选规则 | 红或黑为目标大师，每人最多 100 盘 → 共 **1724 盘** |
| 入选门槛 | 总对局数 ≥ 60（精确保留 20 位大师，剔除数据有偏的偶发对手） |

**关键数据工程**：东萍棋谱的 `[Red]` 字段为"队名+棋手名"（如 `黑龙江 郭莉萍`），
需结合 `[RedTeam]` 标签剥离队名前缀，才能正确按棋手聚合。

处理流程：
```
ICCS棋谱 → 解析(棋手名/胜负/着法序列) → 逐盘重演棋局
        → 提取行为特征 → 按棋手聚合 → 标准化 → PCA降维 + K-Means聚类 → 交互可视化
```

## 三、特征体系（18 维）

**核心思想**：原始数据只有"棋手名 + 着法坐标 + 胜负"，所有特征都通过在虚拟棋盘上
**逐手重演每盘棋**、即时观测（是否吃子、是否将军、是否进入残局）后统计得到。

| 类别 | 特征 | 计算方式 |
|---|---|---|
| 进攻性 | 每局吃子数、每局将军数 | 逐手判定吃子；走子后用规则检测对方是否被将 |
| 节奏 | 平均步数、速胜率(≤40步胜)、长局率(≥100步) | 步数分布统计 |
| 开局偏好 | 中炮率、飞相率、仙人指路率、起马率 | 由红方首着分类 |
| 多样性 | 开局熵 | 开局类型分布的香农熵 |
| 战绩 | 胜率、和棋率、先手胜率、后手胜率 | 分先后手统计胜/和/负 |
| 残局 | 残局率、平均残局长度 | 子力（车马炮）≤4 判为进入残局 |

**将军检测**为自研轻量规则（非引擎）：从被攻击方的"将"出发，沿四正方向（车/炮/对脸将）
与马的八个位置（含蹩腿判定）反向扫描攻击来源。

**得分率说明**：象棋顶尖对局和棋率极高，故采用棋界通用的"得分率 =（胜+0.5×和）/局"
作为战力口径（例：许银川 121 局 44 胜 73 和 4 负，"胜率"仅 36%，但得分率 66.5%）。

---

## 四、降维与聚类

- **标准化**：StandardScaler，消除各特征量纲差异。
- **降维**：PCA 取前两主成分作为地图坐标（可解释、稳定、无 t-SNE/UMAP 的非确定性问题）。
  - 第一主成分（PC1，方差解释 ≈ 33%）：开局熵、和棋率（正）对 吃子数、中炮率（负）→ **"激进 ↔ 稳健"轴**
  - 第二主成分（PC2，方差解释 ≈ 21%）：残局率（正）对 速胜率、后手胜率（负）→ **"善磨残局 ↔ 速攻速决"轴**
- **聚类**：K-Means，簇数 k 由**轮廓系数（silhouette）自动选取**（非人为指定），最终 **k=5，轮廓系数 ≈ 0.22**。
- **流派命名**：每个簇取"偏离全体均值最远的雷达轴"命名（最具区分度），并保证名称唯一。

## 五、可视化与交互设计

| 模块 | 设计 | 对应问题 |
|---|---|---|
| 风格空间地图 | 散点：颜色=流派，大小=全库对局数（活跃度），姓名常显 | Q1/Q2 |
| 势力范围 | 各流派用半透明凸包区域包裹，标注流派名 | Q2 |
| 风格象限轴 | 中心十字 + 四向语义标注（激进↔稳健 / 残局↔速攻） | Q3 |
| 棋手画像（分层） | 流派标签 + 关键词 → 雷达图 → 相似棋手 → 详细数据(折叠) | Q1 |
| 相似棋手网络 | 选中棋手向 Top5 相似者辐射连线，粗细=相似度 | Q4 |
| 双人对比 | 雷达叠加 + 风格距离 + 差异最大特征表 | Q4 |
| 研究发现 | 自动生成的关键结论卡片（默认首页） | 叙事 |
| 主成分载荷 | 底部展示两轴主导特征 | Q3 |

**技术实现**：纯前端（HTML5 Canvas + 原生 JS，无外部库），数据由 Python 预计算后内嵌为
`data.js`，双击即可打开、无需服务器。配仿古纸色调与"楚河汉界"水墨背景体现象棋主题。

## 六、主要发现

1. **存在 5 个稳定风格流派**（轮廓系数证明非偶然）：

   | 流派 | 代表棋手 | 特征 |
   |---|---|---|
   | 速攻速决型 | 王天一、郑惟桐 | 速胜率最高，节奏最快 |
   | 平稳细腻型 | 许银川、谢靖、蒋川、孙勇征、徐天红 | 进攻最克制，少吃子少将军 |
   | 慢工缠斗型 | 胡荣华、赵国荣、李来群、于幼华、汪洋、赵鑫鑫 | 节奏最慢，中盘缠斗 |
   | 传统专精型 | 杨官璘、李义庭、王嘉良 | 开局高度专一（清一色中炮） |
   | 残局控制型 | 吕钦、柳大华、洪智、陶汉明 | 残局率最高，收官见功 |

2. **代际效应显著**：老一辈大师（杨官璘/李义庭/王嘉良）开局熵仅 0.32、中炮率 0.95，
   现代大师开局熵 1.66，印证早期开局体系单一、现代开局百花齐放。

3. **风格极端个例**：许银川处于"稳健—控制"极端（121 局仅负 4），王天一/郑惟桐独立成
   "速攻"群体，与棋迷公认印象吻合，佐证特征体系的有效性。

4. **风格是可识别的个人指纹（模型验证）**：把每位大师对局按时序交替分成两半，用其中一半的
   18 维画像在 20 人中找回其另一半——**Rank-1 准确率 28%（随机基线 5%，5.6×），Rank-3 48%
   （基线 15%，3.2×）**。远超随机说明棋风是稳定、可量化的信号而非噪声，将项目从"可视化"
   提升为"可验证的分析模型"。

---

## 七、创新点

1. 将象棋棋手行为抽象为**可计算的高维风格向量**（18 维，全部可写出明确公式）。
2. 用降维技术构建**棋手风格地图**，把抽象棋风变为可见的空间位置。
3. **数据驱动地自动发现风格流派**（k 由轮廓系数选取，命名由特征偏离度决定）。
4. 实现**风格的可视化表达与多层次交互**：地图、雷达、关系网络、双人对比联动。
5. 将传统棋谱研究与现代数据可视化方法结合。

---

## 八、局限与未来工作

- **特征仅"行为统计"，缺"质量评价"**：当前未用棋力引擎。第二版可引入 Pikafish 对每盘
  逐步打分，加入"失误次数、优势保持率、局面评分波动"等特征，更能区分现代顶尖高手棋风。
- **方差解释有限**：两主成分合计约 54%，地图是有用近似而非精确划分，不宜过度解读细微位置差。
- **样本与年代偏差**：老一辈大师数字化棋谱少（最少仅 134 盘记录），影响其特征稳定性。
- **可扩展**：增加年代/赛事筛选、扩展到更多棋手、引擎特征二期。

---

## 九、技术栈与运行

- **数据处理**：Python（pandas / numpy）；降维聚类：scikit-learn（PCA / KMeans / silhouette）
- **棋谱解析与重演**：自研 ICCS 解析 + 轻量棋盘规则（将军检测、子力统计）
- **前端**：HTML5 Canvas + 原生 JavaScript（零依赖，离线可用）

运行：
```powershell
pip install -r requirements.txt
cd pipeline
python filter_masters.py <棋谱目录> --per 100   # 从大库筛 20 位大师
python build.py --min-games 60                  # 特征→降维→聚类→生成 web/data.js
# 双击打开 web/index.html
```
